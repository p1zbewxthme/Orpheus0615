# Orpheus GitHub Pages site

이 폴더의 파일을 GitHub 저장소 최상위에 업로드하면 됩니다.

## 파일 구조

```text
index.html
orpheus.html
style.css
preview.html
assets/
└─ background.png
```

- `index.html`: 실제 메인 페이지입니다.
- `orpheus.html`: `Orpheus` 텍스트를 클릭하면 이동하는 실제 하위 페이지입니다.
- `style.css`: 배경, 폰트, 모바일 대응 스타일입니다.
- `preview.html`: 로컬에서 바로 열어 볼 수 있는 독립형 미리보기 파일입니다. 배경 이미지가 HTML 안에 포함되어 있어 파일 하나만 열어도 확인할 수 있습니다.
- `assets/background.png`: 실제 GitHub Pages에서 사용하는 배경 이미지입니다.
